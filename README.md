# Crowd Fund (BlockChain)

이더리움 스마트 컨트랙트 서비스 예시

Remix에서 Solidty를 사용해 클라우드 펀딩 컨트랙트를 생성한다.
터미널에서 블록체인 로컬 네트워크를 만들어 Remix와 연동하고 웹 서비스를 제작하였다.
